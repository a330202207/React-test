##  项目结构

###  树状图
```shell
 |----public
 |      `----logs                       # 公共css
 |----src
 |      |----api                        # ajax相关
 |      |----assets                     # 公共资源
 |      |----components                 # 非路由组建
 |      |----config                     # 配置
 |      |----pages                      # 路由组件
 |      |----utils                      # 工具
 |      |----App.js                     # 应用根组件
 |      `----index.js                   # 入口js
```
